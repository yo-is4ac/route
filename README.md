# Routing System

A simple routing system built from scratch.

## Installation

```bash
git clone https://github.com/isaac-rom3ro/routing.git
```

## Run the Project

```bash
cd https://github.com/isaac-rom3ro/routing.git
php -S localhost:8000
```
